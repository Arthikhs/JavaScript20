## 🚀 How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Arthikhs/JS2026.git
   ```
2. Navigate into a topic folder:
   ```bash
   cd 7fuction
   ```
3. Run any `.js` file with Node.js:
   ```bash
   node filename.js
   ```
4. For HTML files, open directly in a browser.
